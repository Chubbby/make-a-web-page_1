Lesson 1: The Basics of the Web and HTML

Basics of the web

The web contains: the web, HTML, URLs, HTTP, and Web apps. Major pieces are your computer + browser (IE, Firefox), internet cloud, and servers. The internet use a protocol of HTTP to send/receive messages. Your computer sends a HTTP request to a web server then the server returns the document coded in HTML which your web browser then displays to your screen. This video does a good job of explaining.

HTML

Hypertext Markup Language is the most common coding used on the web. It has two parts to it, TEXT which is actually the written words you see and MARKUP which gives style to the text. There can be references to images or other pages.

Principle parts of HTML

There are tags which most require an opening tag and a closing tag with content between them, such as Bold or Emphasis. As you can tell the tags can define how the content looks. The tags along with the content are called an element. There are several different tags such as Anchor which references a hyper-link. The Image tag will allow you to define a URL however it doesn't need a closing tag. The Break tag will allow you to add white space to the screen.

Computers are Stupid

Computers are stupid because they interpret code literally. This means small mistake by a programmer can cause huge problems in a program.

Two types of Elements

There are two types of elements inline and block. The block elements have an invisible box around the contents and seen here by the paragraphs.

The Tree Structure of HTML

The tags of HTML will create a tree like structure. This is because there can be multiple tags and elements within another element. Look at this code, there is a DIV tag that has a h2 tag and a P tag. The P tag has several span tags within it. These can be looked at like nested boxes within each other. This is referred as DOM Document Object Model. When you read an HTML document as text, you see a wave of changing indentations going up and down the left side of the document. The more indented an element is, the more deeply nested its corresponding "box" is.

Text Editors Tools

When writing code programmers use special text editors such as Sublime or CodePen. These editors make the coding easier and some will automatically generate a closing HTML tag when you write an opening tag.

Lesson 3: Using CSS to structure HTML Code

What is CSS?

Cascading Style Sheets is what CSS stands for and allows control over style and structure of HTML code. It allows programmer to define style and then refer to it from the HTML code.

CSS avoids Repetition

Programmers try to avoid repetition which waste time and allows for possibility of more errors. CSS will aid in this by allowing style and structure to be defined once and then referencing it multiple times from the code. This will aid in consistency of the document and allow one change to update every reference to the class.
